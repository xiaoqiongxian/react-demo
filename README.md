# react-demo
基于react+react-router+redux+react-redux+antd的web端练习工程

## Requirement
* 安装nodeJs，版本要求6.10.x

## Start the development server:
```bash
npm install
```
```bash
npm start
```
Navigate to [http://loaclhost:3000](http://loaclhost:3000) or [http://127.0.0.1:3000](http://127.0.0.1:3000)

## Build
```bash
npm run build
```
